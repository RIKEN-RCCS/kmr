User Commands kmrckptdump(1)

NAME
kmrckptdump − dump checkpoint log and data

SYNOPSIS
kmrckptdump ckpt-filename

DESCRIPTION
kmrckptdump shows contents of a checkpoint file. A checkpoint file can be a checkpoint log file or a checkpoint data file of each key-value store generated while executing with "ckpt_enable=1".
