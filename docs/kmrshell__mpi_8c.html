Detailed Description

kmrshell_mpi is command line version of KMR and it runs a MapReduce program whose mapper and reducers are user specified MPI programs.

When kmrshell_mpi is used to run a MapReduce program, user should specify a simple program that generates key-value pairs from the output of mapper. The key-value generator program can be specified by '-k' option and can be implemented by reading outputs of mapper and then writing key-value pairs to the standard output. After shuffling the key-value paris, key-value pairs are written to files on each rank with 'key'-named text files whose line represents a key-value separated by a space. The file is passed to the reducer as the last parameter.

kmrshell_mpi can run Map-only MapReduce where no reducer is run. This is very useful if you want to run multiple tasks as a single job.

Options

  • -m mapper [Mandatory]
    • Specify a mapper program (MPI)
  • -k key_value_generator [Optional]
    • Specify a key-value pair generator program (serial)
  • -r reducer [Optional]
    • Specify a reducer program (MPI)
  • -n m_num[:r_num] [Optional]
    • Specify number of MPI processes to run mapper and reducers. When r_num is specified, each mapper runs with m_num processes and each reducer runs with r_num processes. When r_num is not specified each mapper and reducer runs with m_num processes. The default is 2.

Usage

$ mpiexec -machinefile machines -n 4 \
./kmrshell_mpi -n m_num[:r_num] -m mapper [-k kvgenerator] [-r reducer] \
inputfile

Examples

e.g.1) Run MPI mapper and MPI reducer with 2 MPI processes each.
$ mpiexec -np 2 ./kmrshell_mpi -n 2 -m "./mpi_pi.mapper" -k "./mpi_pi.kvgen.sh" -r "./mpi_pi.reducer" ./work
e.g.2) Run MPI mapper with 2 MPI processes and MPI reducer with 4 processes.
$ mpiexec -np 2 ./kmrshell_mpi -n 2:4 -m "./mpi_pi.mapper" -k "./mpi_pi.kvgen.sh" -r "./mpi_pi.reducer" ./work
e.g.3) Only run MPI mapper with 2 MPI processes
$ mpiexec -np 2 ./kmrshell2 -n 2 -m "./mpi_pi.mapper" ./work
