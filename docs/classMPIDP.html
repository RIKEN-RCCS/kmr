Detailed Description

A Tool to Run Tasks under MPI.

MPI-DP runs tasks which are almost independent with master-worker scheduling. It reads a "jobs-list" table, starts MPI processes, and then calls an application entry point. _ARGC and _ARGV are copies of ones passed to main. _APPARGV is the unhandled part of argv which is passed to the application as argv. _TABLE_NUMOF_FIELDS on rank0 holds the number of fields in the jobs-list table.
