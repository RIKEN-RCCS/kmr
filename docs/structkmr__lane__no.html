Detailed Description

Lane Number (at-all-ranks).

Lanes are hierarchical partitioning of communicators, and their depth is limited by KMR_LANE_LEVELS. Currently, the lane-number (structure kmr_lane_no) is stored as unsigned-long and the levels are limited by four. Unspecifed levels must be set to KMR_NO_LANE. If KMR_ANY_LANE appears at a level, it should so at the subordinate levels.
